# 🌲 Sámi Home Recipe Generator

A happy Python project - traditional Sámi recipes from ingredients you already have at home.

### 
The Sámi are the indigenous people of Sápmi (northern parts of Norway, Sweden, Finland, and Russia).  
Traditional Sámi cuisine is simple, sustainable, and deeply connected to nature - based on reindeer, fish, wild berries, herbs, and respect for the nature.

This program helps you discover real Sámi dishes using only what you already have in your kitchen.

### Features
- Enter ingredients one by one (no commas needed)
- Finds full matches and partial matches
- Shows real traditional Sámi recipes with facts and steps
- Saves your cooking history
- Beautiful and friendly console interface

### How to run
1. Download ZIP or clone repo
2. Open Terminal
3. cd into the folder sami-home-recipe-generator
4. python3 main.py
